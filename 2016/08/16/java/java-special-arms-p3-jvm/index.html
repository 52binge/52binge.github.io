2. Java 字节码结构

javac 命令本身只是一个引导器，它引导编译器程序的运行。编译器本身是一个java程序 com.sun.tools.javac.main.JavaCompiler, 该类完成 java 源文件 的 Parser、Annotation process、检查、泛型处理、语法转换等，最终胜出 Class 文件。
