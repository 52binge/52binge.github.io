Cost Function II

Hypothesis $h_{\theta} (x) = \theta_0 + \theta_1 x$
Parameters $\theta_0 、\theta_1$
Cost Function $J(\theta_0，\theta_1) = {\frac {1} {2m}} \sum_{i=1}^m (h_{\theta} (x^{i}) - (y^{i}))^2$
Goal $minimize J(\theta_0，\theta_1)$
