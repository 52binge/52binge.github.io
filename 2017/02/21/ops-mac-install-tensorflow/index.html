3. install virtualenv

virtualenv是python的沙箱工具.我们毕竟是在自己机器上做实验,为了不来回修改各种环境变量,我们一般还是弄个沙箱完比较好.测试完直接删除就行,不用再去改各种配置文件.

安装好后创建一个工作目录,我直接在home里创建了个文件夹.

然后进入目录激活沙箱.
