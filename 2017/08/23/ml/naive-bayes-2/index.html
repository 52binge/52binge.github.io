朴素贝叶斯，适用于新闻分类问题

p(x)p(y∣x)=p(y)p(x∣y)p(x)p(y|x) = p(y)p(x|y) p(x)p(y∣x)=p(y)p(x∣y)

1. 贝叶斯理论简单回顾

在我们有一大堆样本（包含特征和类别）的时候，我们非常容易通过统计得到 p(特征∣类别)p(特征|类别)p(特征∣类别).
大家又都很熟悉下述公式：

p(x)p(y∣x)=p(y)p(x∣y)p(x)p(y|x) = p(y)p(x|y) p(x)p(y∣x)=p(y)p(x∣y)

所以做一个小小的变换

p(特征)p(类别∣特征)=p(类别)p(特征∣类别)p(特征)p(类别|特征) = p(类别)p(特征|类别) p(特征)p(类别∣特征)=p(类别)p(特征∣类别)

p(类别∣特征)=p(类别)p(特征∣类别)p(特征)p(类别|特征) = \frac{p(类别)p(特征|类别)}{p(特征)} p(类别∣特征)=p(特征)p(类别)p(特征∣类别)​

2. 独立假设

看起来很简单，但实际上，你的特征可能是很多维的

p(features∣class)=p(f0,f1,…,fn∣c)p(features|class) = p({f_0, f_1, \ldots ,f_n}|c) p(features∣class)=p(f0​,f1​,…,fn​∣c)

就算是2个维度吧，可以简单写成

p(f0,f1∣c)=p(f1∣c,f0)p(f0∣c)p({f_0, f_1}|c) = p(f_1|c, f_0)p(f_0|c) p(f0​,f1​∣c)=p(f1​∣c,f0​)p(f0​∣c)

加一个牛逼的假设：特征之间是独立的

p(f0,f1∣c)=p(f1∣c)p(f0∣c)p({f_0, f_1}|c) = p(f_1|c)p(f_0|c) p(f0​,f1​∣c)=p(f1​∣c)p(f0​∣c)

其实也就是：

p(f0,f1,…,fn∣c)=Πinp(fi∣c)p({f_0, f_1, \ldots, f_n}|c) = \Pi^n_i p(f_i|c) p(f0​,f1​,…,fn​∣c)=Πin​p(fi​∣c)

3. 贝叶斯分类器

其实我们就是对每个类别计算一个概率 p(ci)p(ci)p(ci) ，然后再计算所有特征的条件概率 p(f_j∣c_i)p(f\_j|c\_i)p(f_j∣c_i) ，那么分类的时候我们就是依据贝叶斯找一个最可能的类别：

p(class_i∣f_0,f_1,…,f_n)=p(class_i)p(f_0,f_1,…,f_n)Πn_jp(f_j∣c_i)p(class\_i|{f\_0, f\_1, \ldots, f\_n})= \frac{p(class\_i)}{p({f\_0, f\_1, \ldots, f\_n})} \Pi^n\_j p(f\_j|c\_i) p(class_i∣f_0,f_1,…,f_n)=p(f_0,f_1,…,f_n)p(class_i)​Πn_jp(f_j∣c_i)
