Pandas Merge   2017-12-31


pandas中的merge和concat类似,但主要是用于两组有key column的数据,统一索引的数据.

依据一组key合并

    A   B key   C   D
0  A0  B0  K0  C0  D0
1  A1  B1  K1  C1  D1
2  A2  B2  K2  C2  D2
3  A3  B3  K3  C3  D3

依据两组key合并

合并时有4种方法how = ['left', 'right', 'outer', 'inner']，预设值how='inner'

Indicator

indicator=True会将合并的记录放在新的一列

   col1 col_left
0     0        a
1     1        b
   col1  col_right

1     1        b        2.0        both

   col1 col_left  col_right indicator_column
