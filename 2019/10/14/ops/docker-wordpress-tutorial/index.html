Docker microservices   2019-10-14


站在 Docker 的角度，Software is the combination of Containers：

  1. 业务逻辑容器
  2. 数据库容器
  3. 储存容器
  4. …

Docker 使得软件可以拆分成若干个标准化容器，然后像搭积木一样组合起来。

这正是微服务（microservices）的思想：

软件把任务外包出去，让各种外部服务完成这些任务，软件本身只是底层服务的调度中心和组装层。
