Python 函数式编程 zip、lambda、map...   2018-01-24


zip、lambda、map…

zip

zip 函数接受任意多个（包括0个和1个）序列作为参数，合并后返回一个tuple列表

[(1, 4), (2, 5), (3, 6)]

[(1, 4), (2, 5), (3, 6)]
0.5 8
1.0 10
1.5 12

lambda

lambda 定义一个简单的函数，实现简化代码的功能，看代码会更好理解。

fun = lambda x,y : x+y, 冒号前的x,y为自变量，冒号后x+y为具体运算

x=4
y=6
10

map

map 是把 函数 和 参数 绑定在一起.

[3]
[4, 6]
