本周我们将要学习 Logistic Regression, 它是神经网络的基础.

Logistic Regression 可以看成是一种只有输入层和输出层(没有隐藏层)的神经网络.

我们将使用 Python 来实现一个这样的模型, 并将其应用在 cat 和 non-cat 的图像识别上.
